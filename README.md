# IntelliStack Platform - GitHub Pages Deployment

This repository hosts the IntelliStack AI-Native Learning Platform for Physical AI & Humanoid Robotics Education, deployed using GitHub Pages.

## Access the Platform

The platform is available at: https://saramali15792.github.io/AINativeBook/

## About

IntelliStack is an advanced learning management system designed specifically for physical AI and humanoid robotics education, featuring:

- Progressive learning paths (5 stages from Foundations to Capstone)
- AI-powered tutoring with Socratic method
- RAG chatbot with citations and source navigation
- Content authoring tools with versioning
- Institution management with cohorts and analytics

## Development

This site is built with Docusaurus and automatically deployed to GitHub Pages when changes are pushed to the main branch.

For more information about the project, see the [docusaurus documentation](./intellistack/content/docs/intro.md).
